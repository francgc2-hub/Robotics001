## ✨ Introducción
Bienvenido a **Robotics01** 🚀 — un repositorio dedicado a guardar y documentar código de **robótica** 🦾 enfocado en:

- 🧭 **Cinemática Directa** (Forward Kinematics)
- 🧩 **Cinemática Inversa** (Inverse Kinematics)
- ⚙️ **Dinámica** (Dynamics)

La mayoría de los ejercicios y proyectos estarán desarrollados **principalmente en Python** 🐍, basados en:

- 📦 **Robotics Toolbox for Python** de **Peter Corke**
- 🧠 **RoboDK API** para simulación e integración
